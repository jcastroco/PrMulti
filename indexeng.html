Login

The login screen allows access to the application. Below is a basic explanation of its functionality:

  • Username: Enter your username in the specified field.
  • Password: Enter your password in the corresponding field.
  • "Ok" Button: Click this button to confirm the data and proceed.
  • "Cancel" Button: Click this button to exit the application if you don't wish to continue.
Note: If the credentials are correct, access will be granted. Otherwise, you will need to verify the entered information.



User Registration

This window allows a user to register in the application by entering their details, which are saved in the database:

  • Email: Field to enter your email.
  • First Name: Field to enter your real first name.
  • Last Name: Field to enter your surname.
  • DNI: Field to enter the DNI (ID number).
  • Alias: Field to enter your alias.
  • Password: Field to enter your password.



Selection Menu

In this window, the user can choose which action to perform with the program. The available options are as follows:

  • FTP System
    • Allows access to the FTP system to manage files (upload, download, or organize content).
  • Email System
    • Grants access to the email system where you can send and receive messages.
  • Settings
    • Allows access to configuration options to adjust program preferences.
  • Company Information
    • Displays information related to the company or program details.



FTP Window

Represents a graphical user interface (GUI) designed for a file management system based on FTP (File Transfer Protocol). This system allows users to manage files and directories on a remote server through displayed options and functionalities.
