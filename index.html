Login

La pantalla de inicio de sesión permite acceder a la aplicación. A continuación, se explica su funcionamiento básico:

  • Username: Escribe tu nombre de usuario en el campo indicado.
  • Contrasena: Introduce tu contraseña en el campo correspondiente.
  • Botón "Ok": Haz clic en este botón para confirmar los datos y continuar.
  • Botón "Cancelar": Si no quieres continuar, haz clic en este botón para cerrar la aplicación.
Nota: Si los datos son correctos, se te permitirá acceder. Si no, tendrás que revisar la información ingresada.



Registro de usuario

Esta ventana hace que un usuario se registre en la aplicación introduciendo sus datos, los cuales se guardan en la base de datos:

  • Correo: Campo para introducir tu correo electronico.
  • Nombre: Campo para introducir tu nombre real.
  • Apellidos: Campo para introducir los apellidos.
  • DNI: Campo para introducir el DNI.
  • Alias: Campo para introducir tu alias.
  • Contrasena: Campo para introducir tu contrasena.



Menu Selección

En esta ventana, el usuario puede elegir qué acción realizar con el programa. Las opciones disponibles son las siguientes:

  • Sistema FTP
    • Permite acceder al sistema FTP para gestionar archivos (subir, descargar o administrar contenido).
  • Sistema de correos
    • Da acceso al sistema de correos donde puedes enviar y recibir mensajes.
  • Configuración
    • Permite acceder a las opciones de configuración para ajustar las preferencias del programa.
  • Información de la empresa
    • Muestra información relacionada con la empresa o detalles sobre el programa.



Ventana FTP

Representa una interfaz gráfica de usuario (GUI) diseñada para un sistema de gestión de archivos basado en FTP (File Transfer Protocol). Este sistema facilita a los usuarios la administración de archivos y directorios en un servidor remoto mediante las opciones y funcionalidades mostradas en pantalla.

Descripción de la Interfaz

  • Botón "Reiniciar": Permite al usuario actualizar la lista de archivos y directorios que se muestran en el árbol de archivos del sistema. Esto garantiza que cualquier cambio realizado, como la creación, eliminación o modificación de archivos, sea reflejado en tiempo real.
  • Caja de texto y Botón "Rename": Facilita la renombración de un archivo o directorio seleccionado. El usuario puede introducir un nuevo nombre en la caja de texto y, al pulsar "Rename", el sistema ejecutará la acción.
  • Árbol de Archivos (JTree): El árbol de archivos muestra la estructura jerárquica de directorios y archivos disponibles en el servidor FTP. En el ejemplo de la imagen, se observan tres carpetas principales:
    • colors
    • sports
    • food
    Estas carpetas pueden contener más archivos o subdirectorios, representando el sistema organizativo del servidor.
  • Botones de Funciones Principales: A la izquierda de la interfaz, hay una serie de botones que facilitan diferentes acciones de gestión de archivos y directorios:
    • Crear directorio: Permite crear un nuevo directorio dentro del servidor FTP.
    • Borrar directorio: Facilita la eliminación de un directorio seleccionado. Es importante asegurarse de que el directorio no contenga archivos antes de borrarlo, dependiendo de las reglas del servidor.
    • Modificar directorio: Permite modificar atributos o configuraciones de un directorio existente, como permisos de acceso.
    • Eliminar archivo: Proporciona la funcionalidad de eliminar un archivo seleccionado del servidor.
    • Descargar archivo: Permite descargar un archivo desde el servidor FTP al sistema local del usuario.
  • Botón "Salir": Finaliza la sesión y cierra la aplicación.

Funcionalidad General del Sistema

Este sistema de gestión FTP proporciona una interfaz amigable que facilita al usuario interactuar con archivos y directorios remotos sin necesidad de usar comandos FTP en una terminal. Entre sus funcionalidades clave se incluyen:
  • Navegación visual: Permite explorar la estructura del servidor FTP de forma intuitiva.
  • Gestión de directorios: Creación, eliminación y modificación de directorios.
  • Administración de archivos: Descarga, eliminación y renombramiento de archivos.
  • Sincronización rápida: La opción "Reload" actualiza la lista de archivos y directorios en tiempo real.

Usabilidad

La interfaz gráfica simplifica el uso del protocolo FTP, especialmente para usuarios que no están familiarizados con herramientas basadas en línea de comandos. Además, proporciona botones claramente etiquetados que permiten ejecutar acciones comunes de manera rápida y eficiente. En resumen, este sistema automatiza y agiliza las tareas de administración de archivos y directorios en un servidor FTP, ofreciendo una solución práctica y visual para la gestión remota de datos.



Ventana Email

La Ventana Email consta de dos ventanas: La primera ventana contiene la gestión de correos del usuario,
donde se pueden usar las siguientes funciones.

  • Nuevo Correo: Crear nuevo correo
  • Bandeja de Entrada: Acceder a la bandeja de entrada de correos
  • No leidos: Accede a la lista de correos sin leer
  • Enviados: Accede a la lista de correos enviados
  • Borradores: Accede a la lista de borradores
  • Spam: Accede a la lista de correos no deseados
  • Basura: Accede a la lista de correos eliminados

La segunda es la ventana de envío,
donde se pueden añadir archivos al correo
  • Para: Destinatario del correo
  • Asunto: Asunto del correo
  • Mensaje: Contenido del correo
  • Anadir Archivo: Boton para anadir un archivo al correo
  • Enviar: Boton para enviar el correo
  • Cancelar: Boton para cancelar el email y volver a la vista anterior



Ventana Configuracion

Ventana configuracion de Usuario Personal: Objetivo: Permite que un usuario configure y modifique su propia información personal dentro del sistema. Campos Disponibles:

  • Alias: El usuario puede establecer o modificar su apodo o alias dentro del sistema.
  • Nombre: Campo donde el usuario escribe su nombre real.
  • Last Name (Apellido): Permite ingresar o actualizar los apellidos del usuario.
  • Correo: Campo destinado a la dirección de correo electrónico personal o de trabajo.
  • Contarseña: Facilita la creación o cambio de la contraseña de acceso.
  • DNI: Sección para el Documento Nacional de Identidad o un número único de identificación.
  • Rol Description (Descripción del Rol): Campo donde se puede especificar el rol o responsabilidad asignada al usuario dentro de la organización.
  • Botones Disponibles:
    • Eliminar: Permite borrar el usuario o eliminar ciertos datos del sistema.
    • Modificar: Opción para editar los campos de información ya existentes.
    • Salir: Cierra la ventana o regresa a la interfaz principal.

Ventana Configuración de Admin para Gestión de Usuarios
  • Estructura de la Tabla:
    • Nombre: Muestra el nombre de cada usuario registrado en la base de datos.
    • Correo: Correo electrónico del usuario para contacto o recuperación de cuenta.
    • Rol: Campo que describe la función específica del usuario dentro de la empresa (por ejemplo: trabajador, supervisor, gerente).
  • Funciones Principales:
    • Crear: Permite agregar nuevos usuarios a la base de datos.
    • Modificar: Facilita la actualización de la información existente de unusuario.
    • Borrar: Elimina a un usuario específico de la base de datos.
    • Volver: Regresa a la interfaz principal o anterior.
